# Python Commit Checks

Time to clean up in preparation for commit. Please perform these tasks:

1.  **Format**: Check for standard python formatters.
    - Try `ruff format .` or `black .`.
2.  **Lint**: Run standard python linters.
    - Try `ruff check .`, `pylint`, or `flake8`.
3.  **Test**: Run tests using standard tools.
    - Try `pytest` or `unittest`.

Wait for all of these things to complete without errors before committing. If there
are errors, fix the errors and try again before committing.
